## 📋 Descripción

Proyecto de **Análisis Exploratorio de Datos (EDA)** desarrollado para 
identificar los factores asociados a la evasión de clientes (*Churn*) 
en la empresa Telecom X.

Los datos fueron extraídos desde una **API pública en formato JSON**, 
procesados mediante un pipeline ETL completo y analizados con 
visualizaciones estratégicas para generar insights accionables.

---

## 🎯 Objetivos

- 📥 Importar datos desde una API REST (JSON)
- 🔧 Aplicar proceso ETL completo
- 📊 Realizar EDA con visualizaciones
- 💡 Generar insights y recomendaciones

## 🚀 Cómo Ejecutar

### Opción 1 — Google Colab (Recomendado)
1. Abre el archivo `Desafio_Telecom_X.ipynb`
2. Clic en el botón **"Open in Colab"**
3. Ejecuta las celdas en orden con `Shift + Enter`
